# XUPT-70MiniProgram
此项目为西安邮电大学70周年校庆微信小程序。

技术栈: `原生JS` `WeUI` 

文件目录：

```
├── README.md
├── image //静态文件，icon等
├── app.js
├── app.json
├── app.wxss
├── pages // 页面存放处
├── thirdparty // 第三方框架，引入WeUI
└── utils // 工具函数
```
